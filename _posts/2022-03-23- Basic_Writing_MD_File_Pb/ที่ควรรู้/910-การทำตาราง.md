## การทำตาราง
 + ใช้รูปแบบการเขียนดังนี้
   ```
   |หัวตาราง1|หัวตาราง2|หัวตาราง3|
   |-------|-------|-------|
   |ข้อมูล1  |ข้อมูล2  |ข้อมูล3  |
   |ข้อมูล4  |ข้อมูล5  |ข้อมูล6  |
   ```
   + ตัวอย่าง
     <table>
      <tr>
       <th>หัวตาราง1</th>
       <th>หัวตาราง2</th>
       <th>หัวตาราง3</th>
      </tr>
      <tr>
       <td>ข้อมูล1</td>
       <td>ข้อมูล2</td>
       <td>ข้อมูล3</td>
      </tr>
      <tr>
       <td>ข้อมูล4</td>
       <td>ข้อมูล5</td>
       <td>ข้อมูล6</td>
      </tr>
     </table>
