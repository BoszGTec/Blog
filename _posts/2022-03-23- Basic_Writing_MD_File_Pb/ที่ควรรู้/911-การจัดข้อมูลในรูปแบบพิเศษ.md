## การจัดข้อมูลในรูปแบบพิเศษ
  + รูปแบบ code ใช้รูปแบบการเขียนดังนี้ <br>
    ''' ภาษาของโค้ด(ไม่เขียนก็ได้) <br>
    โค้ด <br>
    ''' <br>
    \* เปลี่ยน ''' เป็น ```
    + ตัวอย่าง
      ``` py
      for i in range(0,10) :
        print(i)
      ```

      แบบการเขียนของตัวอย่าง
      ---
      ``` 
      '''py
      for i in range(0,10) :
        print(i)
      '''
      ```
      \* เปลี่ยน ''' เป็น ```
      
   + ไฮไลท์ข้อความ ใช้รูปแบบการเขียนดังนี้ <br>
     ''' ข้อความ ''' <br>
     \* เปลี่ยน ''' เป็น ```
     + ตัวอย่าง <br>
       ```ข้อความ```
