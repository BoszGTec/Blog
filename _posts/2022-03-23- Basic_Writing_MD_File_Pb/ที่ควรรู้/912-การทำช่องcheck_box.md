## การทำช่อง check box
  + แบบไม่ใช่ ใช้รูปแบบการเขียนดังนี้ <br>
    ```+ [ ] ข้อความ```
    + ตัวอย่าง
      + [ ] ไม่ใช่
  + แบบใช่ ใช้รูปแบบการเขียนดังนี้ <br>
    ```+ [x] ข้อความ```
    + ตัวอย่าง
      + [x] ใช่
